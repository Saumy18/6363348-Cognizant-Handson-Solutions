## 📌 Overview
This repository covers **Advanced SQL concepts, Unit Testing using NUnit, and Mocking using MOQ**. It demonstrates proficiency in writing optimized SQL queries and building reliable, test-driven C# applications.
---
## 🛠️ Technologies Used
- **SQL Server** – Advanced SQL Querying
- **.NET Core / C#** – Application Development
- **NUnit** – Unit Testing Framework
- **MOQ** – Mocking Framework for Dependency Injection
- **Visual Studio 2022** – Development Environment
---

## 🔍 Project Breakdown

### 📁 Advanced SQL
- Complex SQL Queries
- Window Functions
- Joins and Subqueries
- Aggregations & Performance-Oriented SQL Solutions

### 📁 MoqMailSenderTest
- Demonstrates **Dependency Injection** using Moq
- Unit testing email sender services without real implementations
- Focuses on **mocking external services** for isolated testing

### 📁 NUnitCalculatorTest
- Calculator project with **Test-Driven Development (TDD)**
- Covers test cases for arithmetic operations
- Validates the accuracy and robustness of core functionalities
