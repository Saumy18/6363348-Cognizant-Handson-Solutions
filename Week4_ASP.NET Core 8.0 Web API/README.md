# Week 4 - ASP.NET Core 8.0 Web API Module

## Overview

This module demonstrates the development of RESTful Web APIs using **ASP.NET Core 8.0**, showcasing key backend concepts such as controller routing, model binding, Swagger integration, CRUD operations, and token-based authentication.

It contains five hands-on exercises designed to progressively build understanding and practical experience in API development using modern .NET technologies.

---

## Module Structure

### 1. **First Web API using .NET Core**
- Basic structure of a Web API project
- Introduction to controllers and routing


📂 Directory: `1.First Web Api using .Net core/FirstWebApi`

---

### 2. **Web API using .NET Core with Swagger**
- Swagger/OpenAPI documentation integration
- Annotations for improved API documentation
- Testing endpoints directly from Swagger UI
- Postman
📂 Directory: `2.Web Api using .Net core with Swagger`

---

### 3. **Web API using Custom Model Class**
- Introduction to model classes
- Receiving and returning structured JSON data
- `POST` and `GET` implementation with model binding

📂 Directory: `3.Web Api using custom model class`

---

### 4. **Web API CRUD Operation**
- Full CRUD (Create, Read, Update, Delete) functionality
- In-memory data storage for simulation
- RESTful conventions with proper HTTP status codes

📂 Directory: `4.Web Api CRUD operation`

---

### 5. **Web API with JSON Web Token (JWT)**
- Secure API endpoints using JWT-based authentication
- Token generation and validation
  

📂 Directory: `5.Web Api JsonWebToken`
